전체 사이즈는 620x90. 크기 여유분 확보를 위해, 화면 사이즈보다 좀더 크게 만듦.
위아래 양옆 여유분은 비율이 같게 해주세요!

플레이어의 이동 속도는 초당 40픽셀, 실제 픽셀(사이즈 4배 키운건)160픽셀.
따라서 2초마다 이벤트를 만남. 

이 부분이 루즈하게 느껴지면 애니메이션 프레임을 8->16으로 바꾸고 1초마다 만나게 부탁드립니다


처음 입장 이후, enter 부분에서 해당 스테이지의 제목을 출력.

각 event 부분에 이벤트 오브젝트(현재는 같이 첨부한 임시 이벤트 오브젝트를 사용 부탁드립니다.)가 배치됨.
event와 상호작용 시, 앞의 stop 위치에서 플레이어가 멈추고 상호작용(텍스트 출력)이 시작해야 함.

+ 이번에 새로 추가한 느낌표 이펙트 한번 재생 부탁드립니다. (ani_sheet_32에 추가됨.)

stop 위치는 해당 이벤트에서 35px(4배시 140px) 뒤에 위치함.



g는 게이트가 배치되는 위치.

mg는 메인 게이트가 배치되는 위치
me는 메인 이벤트 오브젝트가 배치되는 위치
